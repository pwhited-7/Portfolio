Basic Nanoprocessor

Circuit Design

For this project I designed a basic nanoprocessor that involved registers, an adder/subtractor, multiplier, TSBA’s, and ROM’s. This design implemented four registers, six TSBA’s, one multiplier, one adder/subtractor, two ROM’s, and a 2:4 decoder. Each of these work simultaneously to perform instructions given by the user. I also utilized all the switches on the FPGA board, two push buttons and three seven-segment displays.

All the components I listed above are connected via bus lines that carry the data bits to where ever the user instructs them to go. The input is only 4 bits because there was only 4 switches I was able to use. I converted this into 8 bits by using a TSBA and grounding 4 additional bits. This circuit takes in a total of 10 different data bits given by Figure 1. One of the ROM’s is a control unit that takes in 4 bits and converts that into a 16 bit output(Figure 2). The data inside the ROM is provided in Figure 3. I only used 11 of these signals to enable or disable the TSBA’s, input and output of the registers, and the adder/subtractor. One of the registers is an accumulator that stores the data received from adding/subtracting or multiplying the given data. The second ROM is used to convert 8 bits into 256 different combinations that is used for the seven-segment display.
